# Local Development Workflow

Use this workflow to set up your local environment, build the project, and start the development server.

## 1. Setup

- [ ] Install dependencies: `pnpm install`.

## 2. Build Verification

// turbo

- [ ] Run the build script to ensure everything is correct: `pnpm run build`.

## 3. Start Server

// turbo

- [ ] Start the local development server: `pnpm run dev`.
