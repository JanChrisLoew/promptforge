# Tech Stack & Dependencies

## Core Framework

- **Runtime:** Node.js >=18.0.0
- **Language:** TypeScript 5.x (Strict Mode enabled)
- **Bundler:** Vite 5.x
- **UI Framework:** React 18.x

## Styling

- **Engine:** Tailwind CSS 3.x
- **Preprocessor:** PostCSS with Autoprefixer
- **Icons:** Lucide React
- **Font:** Mulish (via Google Fonts)

## State & Logic

- **State Management:** React Context or Custom Hooks (`usePromptLibrary`).
- **Persistence:** `window.localStorage`
- **Utilities:** `uuid` for unique ID generation.

## Development Tools

- **Package Manager:** pnpm
- **Linting:** ESLint (implied standard config)
