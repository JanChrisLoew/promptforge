# PromptForge

A professional prompt engineering environment to create, organize, and export AI prompts.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `pnpm install`
2. Run the app:
   `pnpm dev`

> **Note:** This version is standalone and runs locally in the browser. API integration is currently disabled.
